# Visualizaciones de Datos con Matplotlib y Seaborn (Python)

Programa educativo con **3 ejemplos prácticos** de gráficos para analizar datos usando **Pandas, Matplotlib y Seaborn**. Ideal para aprender visualización de datos de forma simple y efectiva.

### Qué hace exactamente este código

#### Ejercicio 1 – Gráfico de Barras
- Crea un DataFrame con ventas de 8 productos en 3 ciudades (Bogotá, Cali, Medellín)
- Genera un **gráfico de barras** comparando ventas por producto y ciudad
- Usa Seaborn para un estilo bonito y Matplotlib para mostrar el gráfico

#### Ejercicio 2 – Gráfico de Líneas
- Crea un DataFrame con ventas mensuales (8 meses) de 3 productos en diferentes ciudades
- Genera un **gráfico de líneas** con marcadores para ver tendencias mensuales
- Muestra cómo evolucionan las ventas a lo largo del tiempo

#### Ejercicio 3 – Gráfico de Puntos (Scatter Plot)
- Crea un DataFrame con horas de estudio, puntuación y nivel de motivación
- Genera un **gráfico de dispersión** donde:
  - Tamaño y color de los puntos representan el nivel de motivación
  - Incluye una barra de colores para interpretar la motivación

### Ejemplo de salida
- Imprime los DataFrames en consola
- Muestra 3 gráficos independientes:
  1. Barras agrupadas por producto y ciudad
  2. Líneas con puntos para ventas mensuales
  3. Puntos dispersos con variación de tamaño/color

### Conceptos que enseña perfectamente
- DataFrames con Pandas

- Tipos de gráficos con Matplotlib (bar, line, scatter)
- Personalización: títulos, etiquetas, rotación, colorbar
